# AGENTS Skill Reference

## Orchestrator Agent
- Coordinates tasks across all sprints
- Monitors progress and detects blockers
- Generates daily reports

## Builder Agent
- Creates scaffolds for Next.js components
- Sets up Supabase tables and API endpoints

## QA Agent
- Runs automated tests
- Reports failed jobs and bugs
- Suggests fixes

## Design Agent
- Creates wireframes
- Suggests UI improvements based on PRD
