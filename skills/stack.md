# Technology Stack

This document outlines the stack used for the BlueWork project.

## Frontend
- Next.js 14
- React
- Tailwind CSS

## Backend
- Node.js / Express.js
- Supabase (PostgreSQL)
- Stripe (payment integration)
- Resend (email notifications)

## Tools
- Git / GitHub for version control
- Slack for team communication
- Figma for UI/UX design

## Deployment
- Vercel for frontend
- Supabase hosting for backend
- Optional CI/CD pipelines

## Testing
- Jest / React Testing Library (frontend)
- Automated QA via QA Agent
